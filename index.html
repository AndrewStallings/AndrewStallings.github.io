Projects

  • Claims Management Software

    Project details - I had a great opportunity to apprentice a professional software engineer, together we worked on a project that would make it easier for insurance adjusters to manage claims they are working on. On this project I helped create key features like the login portal, user input forms, and displaying data from a NoSQL database.

    Technologies - This project uses Typescript, React, and Tailwind CSS for the front-end. The back-end uses firebase. The database uses the NoSQL database provided by firebase.

  • Anti eBay

    Project details - This was my capstone project at Central Washington University. I was on a team with four of my peers and given a client. In our case the project was to create a e-commerce website that is the opposite of eBay where vendors would bid on client requests. We did 7 weeks of programming on this project and 2 weeks of preparation. While the website could use more polish I'm proud of what we accomplished during this timeframe and I learned a lot including React hooks, REST API's, and Jest.

    Technologies - On this project we used a React front-end, Spring back-end, and mariaDB as our database. Jest and JUnit were used for testing.

  • Twitter Scraper

    Project details - This project used the twitter API and python to compare engagement metrics for individual profiles or groups of profiles. The metrics used were follower count, and data the on most recent tweets such as likes and comments. The data was then exported to a spreadsheet.

    Technologies - I completed this project individually using python and the Twitter API to get and manipulate the data. I then stored the results in a CSV file.

  • Todo App

    Project details - I started using a Todo App in 2020 and it was one of the biggest increases ever in my personal productivity. However, the Todo App I had been using became bloated and ad ridden. I decided it would be fun to build one for myself that focused on a minimalist design and features I wanted.

    Technologies - React and Typescript where used to build the application. To convert the web app into a desktop application I used Electron with Electron-React-Boilerplate.
